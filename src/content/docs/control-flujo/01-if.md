## Qué es el controlador If
El controlador If, sirve para saber si realizar una acción o no.

## Ejemplo
```js title="01-if.js"
let edad = 20; // Definimos la variable de edad

if (edad >= 18) { // Si edad es igual o mayor que 18
    console.log("Eres mayor de edad ✅"); 
}
```
